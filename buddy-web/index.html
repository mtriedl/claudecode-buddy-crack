How to Install Your Custom Buddy


    

      
1

      

        
Download the patcher

        
Save buddy-crack.js to any folder. It's a standalone Node.js script — no dependencies.

        
curl -O https://raw.githubusercontent.com/anthropics/claude-code-leaked/main/buddy-crack.js

        
Or download it manually from the repository.

      

    


    

      
2

      

        
Close Claude Code

        
The binary can't be patched while it's running. Exit all Claude Code sessions first.

      

    


    

      
3

      

        
Patch the binary

        
This creates a backup and patches claude.exe to accept custom companion data. Fully reversible.


        

          
          
        

        

          
node buddy-crack.js patch

        

        

          
node buddy-crack.js patch

          
On macOS, the binary is at ~/.claude/local/claude. The patcher auto-detects your platform.

        

      

    


    

      
4

      

        
Paste your companion into config

        
Open your Claude config file and replace (or add) the "companion" field with the JSON you copied.


        

          
          
        

        

          
notepad %USERPROFILE%\.claude.json

        

        

          
nano ~/.claude.json

        

        
Find the "companion" key and replace its value with the JSON you copied. If there's no "companion" key, add it at the top level of the JSON object.

        
Or use the patcher directly: node buddy-crack.js inject '<your-json>'

      

    


    

      
5

      

        
Restart Claude Code

        
Open Claude Code. Your custom companion should appear with the rarity, species, hat, and eyes you chose.

      

    


    

      
↩

      

        
To undo

        
The patcher creates a backup of the original binary. Restore it anytime:

        
node buddy-crack.js unpatch
